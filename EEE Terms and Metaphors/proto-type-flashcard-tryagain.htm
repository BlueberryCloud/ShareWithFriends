Kubernetes Flashcards

What is a Kubernetes Cluster?
A collection of nodes governed by a single control plane that maintains the desired state.
What is a Node?
A physical or virtual machine that runs kubelet and a container runtime, executes pods, and reports status to the API server.
What is a Pod?
The smallest deployable unit in Kubernetes, consisting of one or more containers with a shared network namespace and optional shared storage.
What is a Deployment?
Manages ReplicaSets, performs rolling updates/rollbacks, and ensures replicas stay at the desired version and count.
What is a Service?
Provides a stable ClusterIP/DNS name and load-balances traffic across matching pods using label selectors.
