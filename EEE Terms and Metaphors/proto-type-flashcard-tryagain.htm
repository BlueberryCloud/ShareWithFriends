Kubernetes Flashcards

What is a Kubernetes Cluster?
A collection of nodes governed by a single control plane that maintains the desired state.
What is a Node?
A physical or virtual machine that runs kubelet and a container runtime, executes pods, and reports status to the API server.
What is a Pod?
The smallest deployable unit in Kubernetes, consisting of one or more containers with a shared network namespace and optional shared storage.
What is a Deployment?
Manages ReplicaSets, performs rolling updates/rollbacks, and ensures replicas stay at the desired version and count.
What is a Service?
Provides a stable ClusterIP/DNS name and load-balances traffic across matching pods using label selectors.
What is an Ingress?
Provides HTTP(S) entry into the cluster, with rules managed by an Ingress controller to route requests to Services.
What is a Namespace?
A logical scope within a cluster for organizing resources, applying RBAC, quotas, policies, and limits.
What are ConfigMaps & Secrets?
Objects that inject configuration and sensitive data into pods (env vars or files). Secrets are base64-encoded and can be encrypted at rest.
What is a Volume?
Storage attached to pods, either ephemeral (emptyDir) or persistent (via PVCs/PVs and storage classes).
What is the Control Plane?
Components (API Server, Scheduler, Controllers, etcd) that validate/persist state, assign pods, and reconcile desired vs. actual state.
What is the Kubelet?
Agent on each node that ensures containers are created, healthy, and restarted according to pod specs from the API server.
What is Kube-proxy?
Programs node-level rules so Service virtual IPs route traffic to healthy pod endpoints.
What is CoreDNS?
Resolves Service and Pod DNS names to cluster IPs, integrating with the Kubernetes API.
What is a Horizontal Pod Autoscaler?
Adjusts pod replicas based on metrics (CPU, memory, or custom) within defined bounds.
What is a DaemonSet?
Ensures one (or more) pod runs on every node (e.g., log shippers, monitoring agents, CNI components).
What is a StatefulSet?
Manages stateful apps with stable network IDs, persistent storage per replica, and ordered rollout/termination.
