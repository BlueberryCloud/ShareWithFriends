Use volumes (mount local files)

# Windows PowerShell
docker run -v %CD%/site:/usr/share/nginx/html:ro -p 8080:80 nginx

# macOS/Linux
docker run -v $(pwd)/site:/usr/share/nginx/html:ro -p 8080:80 nginx
I mounted my local site folder into the container so edits show up live. I used %CD% on Windows or $(pwd) on Linux/Mac.
